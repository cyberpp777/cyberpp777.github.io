# 忧郁蓝调

> 雨が止んでも、闇は優しいままだ。

水墨晕染 · 暗黑动漫风 · 个人主页。纯静态 HTML，一键部署，永久免费。

## ✨ 特性

- 🎨 **水墨毛笔字体** — Zhi Mang Xing / Long Cang，五层墨晕渲染
- 🌸 **WebGL 樱花粒子** — 1600 片花瓣实时飘落，Bloom 辉光后处理
- 🖱️ **光标墨迹拖尾** — 鼠标移动留下水墨渐变痕迹
- 💥 **点击樱花炸裂** — 任意点击爆出花瓣雨，5 秒淡入淡出
- 🕐 **左侧实时时钟** — 毛笔大字时间 + 日期
- 🎭 **好感度火柴人** — 点击扇巴掌，好感度逐级解锁
- 📝 **留言板** — localStorage 持久化，支持删除
- 🎵 **网易云音乐播放器** — NeteaseMiniPlayer v2，右上角暗色主题
- 📜 **今日诗词** — 每日自动换古诗词
- 🔒 **彩蛋锁** — Email / GitHub / Twitter 好感度解锁
- 🌑 **暗黑水墨风** — 纯黑白灰调色，毛笔墨渍晕染点缀
