# 🌧 如果那天握住的不是硬币而是勇者的手

忧郁蓝调风格的动漫感个人网站，纯静态 HTML，免费托管。

## 怎么改内容

直接编辑 `index.html`，用记事本或任何编辑器打开：

- **名字**：搜 `忧郁蓝调`，改成你的名字
- **副标题**：搜 `在这里写下你想说的话`
- **关于我**：搜 `这里写下你的自我介绍`，改掉三段话
- **作品**：搜 `作品名称`，每个卡片改图标/标题/描述/标签
- **联系方式**：搜 `href="#"`，把 `#` 换成你的链接
- **页面标题**：`<title>` 标签里的 `忧郁蓝调`

## 免费上线（GitHub Pages）

1. 注册 [GitHub](https://github.com)
2. 新建仓库，名字叫 `你的用户名.github.io`
3. 把 `index.html` 上传到仓库
4. 等 1 分钟，访问 `https://你的用户名.github.io` 就能看到了

## 预览

直接用浏览器打开 `index.html` 即可预览。
